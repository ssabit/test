Flat Management System

Description

  • Saad Ibna Omar Sabit

  • ssabit/flat-management-system

  • December, 2018

A web based application to maintain an apartment by flat owner, caretaker and tenant. User can keep track of flat rent, post notice, request for a service, store tenant information, track who comes and goes out of the apartment, staff management, online and offline messaging. This project deployed in Microsoft Azure Web Apps.
